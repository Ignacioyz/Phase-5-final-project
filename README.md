# Project Template: React/Rails API

## Description
- This project is a portfolio site for an artist & my father. It's purpose is to showcase his work, and attract potentil clientele.


## Requirements

- Ruby 2.7.4
- NodeJS (v16), and npm
- Heroku CLI
- Postgresql


## Setup 
To run app: 
$ npm install 
$ bundle install 
$ rails s 
$ npm start --prefix client


Enjoy!
